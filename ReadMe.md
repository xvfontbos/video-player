# 🎬 Reproductor de Vídeo Personal (PWA)

Un reproductor de vídeo lleuger, elegant i dissenyat per funcionar com una aplicació nativa (PWA). Ideal per a dispositius amb recursos limitats com l'HP Stream 14' amb ChromeOS Flex.

## ✨ Característiques
* **Instal·lable:** Funciona com una aplicació independent gràcies a la tecnologia Progressive Web App (PWA).
* **Mode Offline:** Un cop instal·lat, pots obrir el reproductor sense connexió a internet.
* **Interfície Minimalista:** Disseny enfocat a la visualització amb fons fosc i botons discrets.
* **Formats suportats:** MP4, WebM i MKV (amb còdec H.264 o VP8/VP9).
* **Subtítols .srt:** Carrega un fitxer de subtítols extern i es sincronitzen automàticament.
* **Àudio dual:** Suporta una segona pista d'àudio en fitxer separat o pistes natives del vídeo.
* **Controls de teclat:** Espai = play/pausa · ← → = saltar 10s · ↑ ↓ = volum · F = pantalla completa · S = subtítols.

## 🛠️ Tecnologies utilitzades
* **HTML5/CSS3:** Per l'estructura i el disseny visual.
* **JavaScript:** Per la lògica del reproductor i el Service Worker.
* **Web Audio API:** Per la sincronització de la pista d'àudio alternativa.
* **GitHub Pages:** Per l'allotjament gratuït i segur.
