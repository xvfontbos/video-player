# 📚 Lector d'EPUB Personal (PWA)

Un lector de llibres electrònics en format EPUB, lleuger, elegant i dissenyat per funcionar com una aplicació nativa (PWA). Ideal per a dispositius amb recursos limitats com l'HP Stream 14'amb ChromeOS Flex.

## ✨ Característiques
* **Instal·lable:** Funciona com una aplicació independent gràcies a la tecnologia Progressive Web App (PWA).
* **Mode Offline:** Un cop instal·lat, pots obrir el lector sense connexió a internet.
* **Interfície Minimalista:** Disseny enfocat a la lectura amb fons fosc i tipografia nítida (Georgia).
* **Control Total:** Gestió de capítols, canvi de mida de font i navegació intuïtiva. Recorda la última lectura.

## 🛠️ Tecnologies utilitzades
* **HTML5/CSS3:** Per l'estructura i el disseny visual.
* **JavaScript:** Per la lògica del lector i el Service Worker.
* **ePub.js:** La llibreria principal per renderitzar els fitxers .epub.
* **GitHub Pages:** Per l'allotjament gratuït i segur.
